# Design

## Learning objectives

- Know what is a recursive function.
- Know about stack overflow.
- Know what is a tail-call.
- Know how to use accumulators and helpers to make a function tail-recursive.
- Realize that tail-recursive functions are essentially the same as a loop.

## Out of Scope

- Thunks.

## Concepts

- `recursion`

## Prerequisites

- `loops`
- `stack`
- `floating-point numbers` (necessary for one of the tasks)
