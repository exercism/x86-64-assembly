# Design

## Learning objectives

- Know the syntax for SIMD instructions that operate on integers.
- Know about the lane sizes for packed integers.
- Know about `mov` variants for 128-bit integer values.
- Know how to convert between packed floating-point and packed integer values.
- Know how to sign and zero extend packed integer values.
- Know packed addition and subtraction instructions for integer values, both wrapping and saturating variants.
- Know packed multiplication instructions for integer values and how to choose between upper and lower bits.

## Out of Scope

- SIMD bitwise operations.
- Conditionals, masks and blending.
- Horizontal reduction.
- Cross-lanes operations.
